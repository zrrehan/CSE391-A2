## Overview

This project was developed as part of **CSE 391: Programming for the Internet**.  
The assignment focuses on practicing core JavaScript concepts by building small interactive web applications using HTML, CSS, and JavaScript.

The project contains three separate tasks:

1. Fortune Generator  
2. Stopwatch  
3. To-Do List  

The goal of the assignment is to improve understanding of DOM manipulation, event handling, timing functions, arrays, and data persistence using localStorage.

---

## Tasks Included

### 1. Fortune Generator
- Displays a random fortune message each time the page loads.
- Fortune messages are stored in a JavaScript array.
- Users can change the appearance of the fortune box using buttons (color, border, and font changes).

### 2. Stopwatch
- Stopwatch counts time in multiples of 3 seconds.
- Automatically stops at 30 seconds.
- Includes Start, Stop, and Reset controls.
- Resumes from the last recorded time if stopped before completion.

### 3. To-Do List
- Users can add tasks to a list.
- Tasks can be marked as completed using a checkbox.
- Tasks can be deleted individually.
- Data is stored in localStorage so tasks remain after page refresh.
